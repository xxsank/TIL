### 1-2. JavaScript 기초

```javascript
true || false; // true
true && false; // false
!true;         // false
```
> 논리연산자  || 는 OR 연산, && 는 AND 연산, !는 NOT 연산이다.

> OR 연산은 하나 또는 두 피연산자가 true일 경우 true반환

> AND 연산은 두 피연산자가 true일 경우 true 반환

> NOT 연산은 피연산자의 부울린값을 반대로 바꾼다.
